The one with the tadpole chart.

By Thys
Posted 2022-10-09 Updated 2022-11-19 4 min read

I recently came across a tweet by @Haquafresh  that pointed to a Tadpole chart that is now available in Tableau. With a 100 lines of json code, excluding configuration, it is also available in PowerBI using Deneb.

  Dataset

The dataset is the Tableau Superstore dataset, that can be found at the following link.

  Additional fields created

Profit

1
2

Profit = SUM ( Orders[Profit] )

CY

1
2

CY = CALCULATE( [Profit], 'Date'[Year] = 2021 )

PY

1

PY = CALCULATE( [Profit], 'Date'[Year] = 2020 )

Diff

1
2
3
4
5
6
7

Diff = 
VAR __Diff = [CY] - [PY]

VAR __Check = IF ( __Diff > 0, 1, -1 )

RETURN
    __Check

  Fields used

  • ‘Product’[Sub-Category]
  • CY
  • PY
  • Diff

  Specification

The specification in a nutshell:

* layer
  * rule
  * point
  * text

With a small change to the specification, you can change the circle point to an arrow, as proposed by Steve Wexler.
