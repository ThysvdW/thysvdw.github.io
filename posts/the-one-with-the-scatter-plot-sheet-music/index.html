  Specification

This specification can be enhanced with the song lyrics.¬†Add the lyrics to the Music Notes worksheet, next to the corresponding note.¬†Then add the additional text layer for the lyrics.

The specification in a nutshell:

* layer
  * text (ùÖü note length <= 2)
  * text (ùÖû note length > 2 and note length <= 5)
  * text (__ for notes that is outside of the staff)
  * text (ùÖù note length > 5)
  * text (ùÑÄ where the Note = 'SingleBar')
  * text (ùÑû where the Note = 'Gcleff')
  * text (ùÑ© where the Note = 'Rest')

The challenge is to get the notes to align to where the ‚Äúscatterplot‚Äù points are. Note that some notes has a different angle, which is possible with the ‚ÄúAngle‚Äù attribute to the text mark.
